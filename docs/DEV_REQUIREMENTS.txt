AI 저학년 창의 교실 - 개발 필수 항목
=====================================

1. 환경 변수 (.env)
   - VITE_GEMINI_API_KEY : Gemini API 키. 프런트엔드에서 직접 호출되므로, 배포 시에는 프록시 백엔드 구성이 권장됨.
   - VITE_NANOBANANA_MCP_URL : NanoBanana MCP 스타일 변환 엔드포인트 베이스 URL. 예) https://api.nanobanana.school/mcp
   - VITE_NANOBANANA_MCP_TOKEN : NanoBanana측에서 발급한 토큰. requestStyleTransfer 호출 시 x-nanobanana-token 헤더에 사용.

2. 추가 자료
   - Gemini 고정 프롬프트 문서 : 각 모듈(상상 스파링, AI 글쓰기)의 프롬프트 버전을 별도 관리해야 함.
   - NanoBanana MCP 사양서 : 이미지 업로드 필드명, 지원 스타일 목록, 반환 JSON 스키마.

3. 머신러닝 / 파이썬 (필요 시)
   - Python 3.11
   - PyTorch 2.x (CUDA/cuDNN 여부는 배포 인프라에 맞춰 결정)
   - 커스텀 스타일 변환 모델을 로딩하는 스크립트가 필요할 경우 `ml/style_server.py` 형태로 별도 리포에 구축.

4. 개발 체크리스트
   - mock 모드: .env 값이 비어 있을 경우 Gemini는 에러를 뱉으므로, 실제 API 키를 반드시 세팅하거나 백엔드 프록시로 라우팅.
   - 이미지 저장소: NanoBanana 결과물을 CDN 혹은 S3에 저장한다면 SAS/버킷 권한 정보가 추가로 필요.
